# 上海备孕体检指南

为备孕女性量身定制的全面体检方案网站，涵盖医院推荐、体检套餐、项目清单、免费政策和AI智能助手。

**在线访问**: [https://app-neon-six-62.vercel.app](https://app-neon-six-62.vercel.app)

## 快速开始

```bash
cd app
npm install
npm run dev
```

## 技术栈

- React 19 + TypeScript + Vite 7
- shadcn/ui + Tailwind CSS v3
- Supabase Edge Functions + PostgreSQL
- Vercel 部署

## 项目结构

```
health-agent/
├── app/                 # 前端应用
│   ├── src/

│   └── README.md        # 详细文档
├── screenshots/         # 网站截图
└── CLAUDE.md            # 开发指南
```

## 功能特色

- 智能年龄推荐（25-40岁）
- AI备孕顾问聊天助手
- 上海8家优质医院推荐
- 30+体检项目清单
- 免费孕前检查政策指南
- 医学术语词典
